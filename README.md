# Expression Maps

A regularly updated collection of Alex Fortunato's personal expression maps for use in Cubase.

## Compatibility

| Library | Cubase Version |
|---|---|
| All libraries (except VSL Duality Strings) | Cubase 14+ |
| VSL Duality Strings | Cubase 15+ |

VSL Duality Strings expression maps make use of new Expression Map features introduced in Cubase 15.

## Contents

- **Brass** — BBC Symphony Orchestra, Zodiac Brass
- **Strings** — BBC Symphony Orchestra Pro, VSL Duality Strings
- **Woodwinds** — BBC Symphony Orchestra

## Roadmap

More expression maps will be added over time, with the goal of full Cubase 15 compatibility across all libraries.
